# Patrimônio Portões

Site estático para apresentar serviços de serralheria, trabalhos realizados e contato por WhatsApp.

## Como testar

Abra o arquivo `index.html` no navegador.

## O que trocar antes de publicar

1. O nome da serralheria já está como `Patrimônio Portões`.
2. Troque todos os links `https://wa.me/5500000000000` pelo número real no formato `55DDDNUMERO`.
3. Atualize `WhatsApp: (00) 00000-0000` na seção de contato.
4. Troque as imagens de exemplo por fotos reais dos portões, grades, corrimãos e serviços realizados.
5. Ajuste a cidade e região de atendimento.

## Publicação recomendada

Para esse tipo de site, as opções mais simples são GitHub Pages, Vercel ou Netlify.
Como o projeto não precisa de banco de dados nem servidor, ele pode ser publicado como site estático.
